# Faucet

Get testnet ETH and stablecoins for development.

## Get Testnet ETH

Visit the [Tempo Faucet](https://faucet.testnet.tempo.xyz) to request testnet ETH.

You can request up to 0.1 ETH every 24 hours per address.

## Get Test Stablecoins

Test stablecoins are available on the testnet:

- **AlphaUSD** - A test stablecoin for development
- **BetaUSD** - Another test stablecoin
- **PathUSD** - Test stablecoin with fee enabled
- **ThetaUSD** - Test stablecoin with rewards

Use the faucet or mint directly using the SDK.
