# Library Setup

Throughout the documentation, we will be using the libraries below for examples.

## Libraries

### Viem + Tempo extesion (TypeScript)

Viem is a TypeScript library for interacting with EVM-compatible blockchains such as Tempo.

:::note
Please request access to the `tempo-ts` library by reaching out to the Tempo team.
:::

::::steps

#### Install

:::code-group

```bash [npm]
npm i viem tempo.ts
```

```bash [pnpm]
pnpm i viem tempo.ts
```

```bash [bun]
bun i viem tempo.ts
```

:::
