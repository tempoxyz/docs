# Tempo Documentation

Tempo is a Layer 1 blockchain purpose-built for global payments. Unlike general-purpose blockchains, every aspect of Tempo is optimized for the specific needs of real-world payment systems: low and predictable costs, stablecoin flexibility, built-in compliance, and payment-grade user experience.
