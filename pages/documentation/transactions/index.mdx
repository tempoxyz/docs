# Sending Transactions

Tempo provides flexible transaction capabilities designed specifically for payment use cases.

| Topic | Description |
|-------|-------------|
| [**Fee Tokens**](/documentation/transactions/fee-tokens) | Pay transaction fees in any supported USD stablecoin |
| [**Payment Lanes**](/documentation/transactions/payment-lanes) | Guaranteed blockspace for payment transactions |
| [**Batch Transactions**](/documentation/transactions/batch-transactions) | Execute multiple operations atomically |
| [**Fee AMM**](/documentation/transactions/fee-amm) | Provide liquidity to enable fee conversions |
| [**Fee Sponsorship**](/documentation/transactions/fee-sponsorship) | Enable gasless transactions with fee payers |
| [**Account Abstraction**](/documentation/transactions/account-abstraction) | Native AA transaction type with passkeys, parallel nonces, and gas sponsorship |
