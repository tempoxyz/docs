# Tempo Improvement Proposals (TIPs)

TIPs are design documents describing new features and changes to Tempo.

## Active TIPs

| TIP | Title | Status |
|-----|-------|--------|
| TIP-20 | Token Standard | Final |
| TIP-403 | Compliance Policies | Final |
| TIP-1559 | Fee Market | Final |

## TIP Process

1. **Draft** - Initial proposal
2. **Review** - Community feedback
3. **Final** - Accepted and implemented
4. **Deployed** - Live on network

## Submit a TIP

1. Fork the [Tempo repository](https://github.com/tempoxyz/tempo)
2. Create a new TIP document
3. Submit a pull request
4. Participate in review

## TIP Template

```markdown
---
tip: <number>
title: <title>
author: <name>
status: Draft
---

# Abstract

Brief description of the proposal.

# Motivation

Why is this change needed?

# Specification

Technical details of the implementation.

# Rationale

Design decisions and alternatives considered.
```
