# TIP-4217 [Currency Metadata Interface]

Provides a currency metadata interface to standardize representations of stablecoins pegged to the same currency, focusing on decimal precision.

A Solidity reference implementation can be found on Github [here](https://github.com/tempoxyz/specs/blob/main/src/TIP4217Registry.sol) however the full functionality is implemented in Rust as part of `tempoxyz/tempo`.

## Overview

TIP-4217 standardizes decimal precision for ISO 4217-style currency codes, ensuring consistent representation of stablecoins across the Tempo ecosystem.

TIP-4217 is intended to enable the following kinds of use cases:
- **Stablecoin Standardization**: Ensures all tokens pegged to the same currency use consistent decimal precision
- **Cross-Token Compatibility**: Enables seamless interaction between different stablecoin implementations
- **Currency Metadata**: Provides standardized way to query currency-specific information
