# What is Tempo?

Tempo is a purpose-built blockchain optimized for stablecoin applications.

## Key Features

### Native Stablecoin Issuance
Issue stablecoins directly on-chain with TIP-20, including:
- Minting and burning
- Transfer fees
- Holder rewards
- Compliance controls

### Stablecoin DEX
Built-in exchange for trading stablecoins:
- Low slippage between stable pairs
- AMM and orderbook hybrid
- Fee sharing for LPs

### Stablecoin Fee Payment
Pay transaction fees in stablecoins:
- No ETH required for users
- Fee AMM handles conversion
- Better UX for end users

### Fast Finality
- 1-second block times
- Instant confirmation
- Parallel transaction execution

## Use Cases

- Payment applications
- Remittances
- Trading platforms
- Embedded finance
- Micro-transactions
