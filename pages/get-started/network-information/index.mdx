# Network Information

Essential details for connecting to and interacting with Tempo's testnet.

## Current Testnet

We're currently running **Andantino**, which includes:

- Stablecoin Exchange orderbook
- Native account abstraction with WebAuthn support
- Multi-stablecoin fee payments
- Payment lanes for optimized routing
- Sub-second finality with Simplex consensus
