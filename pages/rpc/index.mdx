# Overview

## RPC Methods

| Method | Description |
|--------|-------------|
| **AMM** | |
| [`amm_getLiquidityPools`](/rpc/amm_getLiquidityPools) | Gets paginated liquidity pools from the Fee AMM |
| **Ethereum** | |
| [`eth_getTransactions`](/rpc/eth_getTransactions) | Gets paginated transactions with support for all Ethereum transaction types plus Account Abstraction (0x76) |
| **Policy** | |
| [`policy_getAddresses`](/rpc/policy_getAddresses) | Gets paginated addresses in a transfer policy |
| **Stablecoin Exchange** | |
| [`dex_getOrderbooks`](/rpc/dex_getOrderbooks) | Gets paginated orderbook metadata from the Stablecoin Exchange |
| [`dex_getOrders`](/rpc/dex_getOrders) | Gets paginated orders from the Stablecoin Exchange orderbook |
| **Token** | |
| [`token_getRoleHistory`](/rpc/token_getRoleHistory) | Gets paginated role change history for TIP-20 tokens |
| [`token_getTokens`](/rpc/token_getTokens) | Gets paginated TIP-20 tokens on Tempo |
| [`token_getTokensByAddress`](/rpc/token_getTokensByAddress) | Gets paginated TIP-20 tokens associated with an account address |
