# `eth_getTransactions`

## Overview

Gets paginated transactions on Tempo with flexible filtering and sorting.

Uses cursor-based pagination for stable iteration through transactions.

## Request

```ts
type Request = {
  method: 'eth_getTransactions'
  params: [{
    /** 
     * Cursor for pagination (based on blockNumber:transactionIndex). 
     * 
     * Defaults to first entry based on the sort and filter configuration.
     * Use the `nextCursor` in response to get the next set of transactions.
     */
    cursor?: string
    /** Filter configuration. */
    filters?: {
      /** Filter by sender address (from) */
      from?: Address
      /** Filter by recipient address (to) */
      to?: Address
      /** Transaction type */
      type?: Hex
    }
    /** 
     * Maximum number of transactions to return.
     * 
     * Defaults to 10.
     * Maximum is 100.
     */
    limit?: number
    /** Sorting configuration */
    sort?: {
      /** Field to sort by (e.g., 'blockNumber', 'transactionIndex', 'gasPrice', 'value', 'nonce') */
      on: string
      /** Sort direction */
      order: 'asc' | 'desc'
    }
  }]
}
```

## Response

```ts
type Response = {
  /** Cursor for next page, null if no more results */
  nextCursor: string | null
  /** Array of transactions matching the query */
  transactions: Transaction[]
}

type Transaction = {
  /** Access list */
  accessList?: AccessListEntry[]
  /** Authorization list */
  authorizationList?: AuthorizationEntry[]
  /** Block hash */
  blockHash: Hex
  /** Block number */
  blockNumber: Hex
  /** Block timestamp */
  blockTimestamp: Hex
  /** Blob versioned hashes */
  blobVersionedHashes?: Hex[]
  /** Array of calls to execute */
  calls?: Call[]
  /** Chain ID */
  chainId: Hex
  /** Optional fee payer signature for sponsored transactions */
  feePayerSignature?: Signature
  /** Optional fee token preference */
  feeToken?: Address
  /** Sender address */
  from: Address
  /** Gas limit */
  gas: Hex
  /** Gas price */
  gasPrice: Hex
  /** Transaction hash */
  hash: Hex
  /** Input data */
  input?: Hex
  /** Maximum fee per blob gas */
  maxFeePerBlobGas?: Hex
  /** Maximum total fee per gas */
  maxFeePerGas?: Hex
  /** Maximum priority fee per gas */
  maxPriorityFeePerGas?: Hex
  /** Nonce */
  nonce: Hex
  /** Nonce key for 2D nonce system */
  nonceKey?: Hex
  /** ECDSA signature r */
  r?: Hex
  /** ECDSA signature s */
  s?: Hex
  /** Signature (AA) */
  signature?: Signature
  /** Recipient address */
  to: Address
  /** Transaction index in block */
  transactionIndex: Hex
  /** Transaction type */
  type: Hex
  /** ECDSA recovery id */
  v?: Hex
  /** Transaction valid after this timestamp */
  validAfter?: Hex
  /** Transaction valid before this timestamp */
  validBefore?: Hex
  /** Transfer value */
  value?: Hex
  /** Y parity of signature */
  yParity?: Hex
}
```

## Examples

### Basic pagination

```ts twoslash
// @noErrors

// First page
const response = await provider.request({
  method: 'eth_getTransactions',
  params: [{
    limit: 100
  }]
})
// @log: { nextCursor: '0x...',transactions: [{ type: '0x2',hash: '0xabc...',from: '0x123...',to: '0x456...',value: '0xde0b6b3a7640000',blockNumber: '0x1a2b3c',...}] }

// Next page
const nextResponse = await provider.request({
  method: 'eth_getTransactions',
  params: [{
    cursor: response.nextCursor,
    limit: 100
  }]
})
// @log: { nextCursor: '0x...',transactions: [{ type: '0x76',hash: '0xdef...',from: '0x789...',calls: [{ to: '0xabc...',value: '0x0',input: '0x...' }],...}] }
```

### Filter by address

```ts twoslash
// @noErrors
const response = await provider.request({
  method: 'eth_getTransactions',
  params: [{
    filters: {
      from: '0x1234567890123456789012345678901234567890'
    },
    limit: 50
  }]
})
// @log: { nextCursor: null, transactions: [{ type: '0x2',hash: '0x...',from: '0x1234567890123456789012345678901234567890',to: '0x...',value: '0xde0b6b3a7640000',blockNumber: '0x1a2b3c',...}] }
```

### Filter by transaction type

```ts twoslash
// @noErrors
// Get only AA transactions
const response = await provider.request({
  method: 'eth_getTransactions',
  params: [{
    filters: {
      type: '0x76' // Account Abstraction transactions
    },
    limit: 50
  }]
})
// @log: { nextCursor: null, transactions: [{ type: '0x76',hash: '0x...',from: '0x...',calls: [{ to: '0x...',value: '0x0',input: '0x...' }],feeToken: '0x20c0000000000000000000000000000000000001',nonceKey: '0x0',nonce: '0x5',...}] }
```

### Filter by block range

```ts twoslash
// @noErrors
const response = await provider.request({
  method: 'eth_getTransactions',
  params: [{
    filters: {
      blockNumber: ['0x100000', '0x100fff'],  // Blocks 1048576 to 1052671
    },
    limit: 100,
  }]
})
// @log: { nextCursor: '0x...',transactions: [{ type: '0x2',hash: '0x...',value: '0x52b7d2dcc80cd2e4000000',blockNumber: '0x1000ab',...}] }
```

### Sort by timestamp

```ts twoslash
// @noErrors
const response = await provider.request({
  method: 'eth_getTransactions',
  params: [{
    limit: 20,
    sort: {
      on: 'blockTimestamp',
      order: 'desc'
    }
  }]
})
// @log: { nextCursor: null, transactions: [{ type: '0x2',hash: '0x...',gasPrice: '0x3b9aca00',maxPriorityFeePerGas: '0x59682f00',maxFeePerGas: '0x77359400',...}] }
```
