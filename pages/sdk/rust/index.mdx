# Getting Started

Tempo distributes a Rust SDK in the form of an [Alloy](https://alloy.rs) extension,
a popular Rust crate for EVM-compatible blockchains.

The Tempo extension can be used to perform common operations with the chain, such as:
 querying the chain, sending Tempo transactions, managing tokens & their AMM pools, and more.

## Steps

::::steps

### Install

To install the Tempo extension, you will need to install [Alloy](https://alloy.rs) and Tempo:


```bash [cargo]
cargo add alloy
cargo add tempo-alloy --git https://github.com/tempoxyz/tempo
```

### Configure

Next, we will configure an Alloy provider.

```rust twoslash [main.rs]
use alloy::{
    providers::{Provider, ProviderBuilder},
    transports::TransportError,
};
use tempo_alloy::TempoNetwork;

async fn build_provider() -> Result<impl Provider<TempoNetwork>, TransportError> {
    ProviderBuilder::new_with_network::<TempoNetwork>()
        .connect(&std::env::var("RPC_URL").expect("No RPC URL set"))
        .await
}
```

### Use Actions

Now you're ready to start interacting with Tempo!


```rs twoslash [main.rs]
use alloy::{
    providers::{Provider, ProviderBuilder},
    transports::TransportError,
};
use tempo_alloy::TempoNetwork;

async fn build_provider() -> Result<impl Provider<TempoNetwork>, TransportError> {
    ProviderBuilder::new_with_network::<TempoNetwork>()
        .connect(&std::env::var("RPC_URL").expect("No RPC URL set"))
        .await
}

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let provider = build_provider().await?;

    println!("{}", provider.get_block_number().await?);
    // @log: 421045
    Ok(())
}
```
