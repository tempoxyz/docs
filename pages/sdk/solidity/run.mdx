# Build, Test & Deploy

Build your contracts:

```bash
forge build
```

Run all tests against Tempo's testnet:

```bash
forge test
```

## Deploy Your Contract

:::info
This assumes you have set up your repo with `forge init --tempo` as shown above.
:::

:::note
We are using the `--interactive` flag to prompt for your private key securely at runtime.
:::

### Using `forge create`

Deploy a simple contract to Tempo's testnet:

```bash
forge create src/Counter.sol:Counter \
  --rpc-url https://rpc.testnet.tempo.xyz \
  --interactive \
  --broadcast
```

Deploy a contract with constructor arguments to Tempo's testnet:

```bash
forge create src/Mail.sol:Mail \
  --rpc-url https://rpc.testnet.tempo.xyz \
  --interactive \
  --broadcast \
  --constructor-args <ARGUMENTS>
```

### Using `forge script`

Run a deployment script to deploy to Tempo's testnet:

```bash
forge script script/Mail.s.sol \
  --interactives 1 \
  --sender <YOUR_WALLET_ADDRESS> \
  --broadcast
```

## Verify Your Contract

Check that your contract has been deployed on Tempo's testnet:

```bash
cast code <CONTRACT_ADDRESS> --rpc-url https://rpc.testnet.tempo.xyz
```

Interact with your contract on Tempo's testnet:

```bash
cast call <CONTRACT_ADDRESS> "myFunction()(uint256)" \
  --rpc-url https://rpc.testnet.tempo.xyz
```

Send a transaction to your contract on Tempo's testnet:

```bash
cast send <CONTRACT_ADDRESS> "myFunction(uint256)" 42 \
  --rpc-url https://rpc.testnet.tempo.xyz \
  --interactive
```

Replay a transaction by hash on Tempo's testnet:

```bash
cast run <TX_HASH> \
  --rpc-url https://rpc.testnet.tempo.xyz
```
