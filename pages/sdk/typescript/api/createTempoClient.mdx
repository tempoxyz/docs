# `createTempoClient`

Creates a Tempo client with the `tempo` chain and all [Tempo-specific actions](/sdk/typescript/api#actions) pre-configured.

## Usage

```ts
import { createTempoClient } from 'tempo/viem'
import { http } from 'viem'

const credentials = `${process.env['TEMPO_USERNAME']}:${process.env['TEMPO_PASSWORD']}`

const client = createTempoClient({
  transport: http(undefined, {
    fetchOptions: {
      headers: {
        Authorization: `Basic ${btoa(credentials)}`,
      },
    },
  }),
})
```

:::tip

If you wish to extend an existing Viem client with Tempo-specific actions, you can use the `.extend` method.

```ts
import { createClient, http } from 'viem'
import { tempo } from 'tempo/chains'
import { tempoActions } from 'tempo/viem'

const client = createClient({
  chain: tempo,
  // ...
}).extend(tempoActions())
```

:::

### Custom Chain

You can specify a custom chain (e.g. another Tempo testnet).

```ts
import { createTempoClient } from 'tempo/viem'
import { tempoLento } from 'tempo/chains' // [!code focus]
import { privateKeyToAccount } from 'viem/accounts'

const client = createTempoClient({
  account: privateKeyToAccount('0x...'),
  chain: tempoLento, // [!code focus]
})
```

### Custom Transport

You can specify a custom transport (defaults to `http`).

```ts
import { createTempoClient } from 'tempo/viem'
import { webSocket } from 'viem' // [!code focus]

const client = createTempoClient({
  transport: webSocket('wss://ws-lento.tempoxyz.dev'), // [!code focus]
})
```
