# `fee.getUserToken`

Gets the user's default fee token preference. [Learn more about fee tokens](/documentation/transactions/fee-tokens)

## Usage

:::code-group

```ts twoslash [example.ts]
import { client } from './viem.config'

const { address, id } = await client.fee.getUserToken()

console.log('Fee token address:', address)
// @log: Fee token address: 0x20c0000000000000000000000000000000000000
console.log('Fee token ID:', id)
// @log: Fee token ID: 0n
```

```ts twoslash [viem.config.ts] filename="viem.config.ts"
// [!include ~/snippets/viem.config.ts:setup]
```

:::

## Return Type

```ts
type ReturnType = {
  /** Address of the fee token */
  address: Address
  /** ID of the fee token */
  id: bigint
}
```
