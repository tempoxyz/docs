# Getting Started

Tempo distributes a TypeScript SDK in the form of a [Viem](https://viem.sh) extension, 
a popular TypeScript interface for EVM-compatible blockchains.

The Tempo extension can be used to perform common operations with the chain, such as: 
 querying the chain, sending Tempo transactions, managing tokens & their AMM pools, and more.

## Steps

::::steps

### Install

To install the Tempo extension, you will need to install [Viem](https://viem.sh) and Tempo:

:::code-group

```bash [npm]
npm i viem tempo.ts
```

```bash [pnpm]
pnpm i viem tempo.ts
```

:::
