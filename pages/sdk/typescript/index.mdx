# TypeScript SDK

The `tempo.ts` package provides TypeScript utilities for Tempo.

## Installation

```bash
npm install tempo.ts viem wagmi
```

## Quick Start

```typescript
import { createTempoClient } from 'tempo.ts'
import { tempoTestnet } from 'tempo.ts/chains'

const client = createTempoClient({
  chain: tempoTestnet,
})

// Read token balance
const balance = await client.readContract({
  address: tokenAddress,
  abi: erc20Abi,
  functionName: 'balanceOf',
  args: [userAddress],
})
```

## Packages

| Package | Description |
|---------|-------------|
| `tempo.ts` | Core client and utilities |
| `tempo.ts/chains` | Chain definitions |
| `tempo.ts/hooks` | React hooks for Wagmi |
| `tempo.ts/abi` | Contract ABIs |
| `tempo.ts/server` | Server-side utilities |
