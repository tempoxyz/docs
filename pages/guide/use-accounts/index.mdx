# Create & Use Accounts

Tempo supports multiple account types for different use cases.

## Account Types

### Smart Accounts (Recommended)

Smart accounts with passkey authentication provide:
- Passwordless login with biometrics
- Social recovery options
- Gas abstraction
- Batch transactions

### EOA Wallets

Traditional Ethereum wallets work on Tempo:
- MetaMask, Rainbow, Coinbase Wallet
- Hardware wallets
- Any EVM-compatible wallet
