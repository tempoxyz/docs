# Issue Stablecoins

Tempo provides native stablecoin issuance with built-in compliance features.

## TIP-20 Tokens

TIP-20 is Tempo's native token standard, extending ERC-20 with:

- **Issuer controls** - Mint, burn, freeze, and recover tokens
- **Transfer fees** - Optional fees on transfers
- **Holder rewards** - Distribute yield to token holders
- **Compliance policies** - TIP-403 policy enforcement
