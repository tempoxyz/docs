# Fee Payment Options

Tempo offers flexible fee payment options.

## Pay in ETH (Default)

Standard EVM transactions pay fees in ETH:

```typescript
const { writeContract } = useWriteContract()

writeContract({
  address: tokenAddress,
  abi: erc20Abi,
  functionName: 'transfer',
  args: [recipient, amount],
  // Fee automatically paid in ETH
})
```

## Pay in Stablecoins

Use Tempo Transactions to pay fees in any supported stablecoin:

```typescript
import { useTempoTransaction } from 'tempo.ts/hooks'

const { send } = useTempoTransaction()

send({
  to: tokenAddress,
  data: transferData,
  feeToken: usdcAddress, // Pay fee in USDC
})
```

## Fee Estimation

Estimate fees before sending:

```typescript
import { estimateTempoFee } from 'tempo.ts'

const fee = await estimateTempoFee({
  to: tokenAddress,
  data: transferData,
  feeToken: usdcAddress,
})

console.log('Fee:', fee.amount, fee.token)
```
