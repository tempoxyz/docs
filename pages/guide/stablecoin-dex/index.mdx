# Exchange Stablecoins

Tempo includes a native stablecoin DEX for efficient trading.

## Features

- **Low slippage** - Optimized for stablecoin pairs
- **Instant settlement** - Same-block execution
- **AMM + Orderbook** - Hybrid liquidity model
- **Fee sharing** - Earn from providing liquidity
