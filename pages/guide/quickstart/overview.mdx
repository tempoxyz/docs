## What's Different?

Tempo extends EVM with payment-specific features:
- **Payment lanes**: Dedicated blockspace for payments
- **Fee flexibility**: Pay fees in any stablecoin
- **Account abstraction**: Native support for passkeys and WebAuthn
- **TIP-20 tokens**: Enhanced token standard with compliance features
- **Instant finality**: One-second transaction confirmation

Ready to dive deeper? Explore [How does Tempo work?](/documentation) to understand the protocol in detail.
