# Node Operators

## Node Types

| Type | Purpose | Consensus | Use Case |
|------|---------|-----------|----------|
| **RPC Node** | API access | No | dApps, explorers, wallets |
| **Validator** | Block production | Yes | Consensus participation (if you are interested, contact the Tempo team) |

## Quick Start

1. [System Requirements](/operators/system-requirements) - Hardware specs
2. [Installation](/operators/installation) - Download binary
3. [RPC Node](/operators/rpc-node) - Run a full node
