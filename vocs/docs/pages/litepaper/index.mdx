# Litepaper [The blockchain for payments]

## Abstract 

Tempo is a layer-1 blockchain purpose-built for payments, developed in partnership with category-leading companies to bring global payments volume onchain. Optimized engineering and dedicated blockspace for payments ensure low-cost transfers even when demand is high. A built-in system for stablecoin liquidity allows easy cross-stablecoin payments, as well as paying transaction fees in any stablecoin. Tempo is designed to be a neutral platform with a distributed validator set and permissionless issuance of any stablecoin.
