# Tokens

TIP-20 tokens are designed for real-world payments with built-in features like role-based access control, transfer policies, supply management, and memo support. They integrate seamlessly with Tempo's fee system and compliance requirements.


| Task | Description |
|------|-------------|
| **[Deploy a Token](/documentation/tokens/deployment)** | Create a new TIP-20 token using the factory contract |
| **[Set Up Roles](/documentation/tokens/roles)** | Configure who can mint, burn, and manage your token |
| **[Manage Supply](/documentation/tokens/minting)** | Mint new tokens and burn existing ones |
| **[Configure Policies](/documentation/tokens/policies)** | Control who can transfer your tokens |

For complete technical specifications, see the [Protocol Reference](/protocol/tokens).
