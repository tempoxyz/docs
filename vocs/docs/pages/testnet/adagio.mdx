# Adagio (Testnet 1)

## Release Notes

Adagio is the first release of a proof-of-concept of early Tempo features.

We started the Adagio network on September 3rd, 2025.

The main features introduced were the initial implementations of token standards on Tempo and the option to pay for transactions using a single TestUSD stablecoin.

| Feature                        | Spec                                                                 | Guides                                                                                 |
|---------------------------------|----------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| TIP-20, TIP-403, TIP-4217       | [TIP-20](/protocol/tokens/tip-20), [TIP-403](/protocol/tokens/tip-403), [TIP-4217](/protocol/tokens/tip-4217) | [Token Docs](/documentation/tokens) |
| Pay fees in one stablecoin      | [Fees Spec](/protocol/transactions/fee-amm)                                         | [Transaction Fees](/documentation/transactions/fees)                                   |

## Connecting to Network

Adagio (Adagio) network details and connection information.

| Network Parameter  | Value                                 |
|--------------------|---------------------------------------|
| **Network Name**   | Adagio (Adagio)              |
| **Chain ID**       | `42424`                               |
| **RPC URL**        | `https://rpc-adagio.tempoxyz.dev`     |

Speak to the Tempo team for an RPC username and password.

### Example

To connect to the Adagio (Adagio) and retrieve the latest block information using `cast`, you can use the following command:

```sh
cast block \
   --rpc-url https://${RPC_USERNAME}:${RPC_PASSWORD}@rpc-adagio.tempoxyz.dev
```

## Testnet Tokens

There is a single token available on the first testnet for use in paying gas fees.

### TestUSD

The gas token for Adagio. Future testnets will allow paying gas in any USD stablecoin.

| Parameter  | Value                                      |
|------------|--------------------------------------------|
| **Address**| `0x20c0000000000000000000000000000000000000`|
| **Symbol** | TestUSD                                    |
| **Decimals**| 18                                        |

Get TestUSD tokens using our builtin faucet for the testnet.

```bash
cast rpc tempo_fundAddress 0x820525B56cF3cD426e58507F65DCaCB90BceEFb8
```

### Using the Faucet

Get TestUSD tokens using the built-in faucet:

```bash
# Fund your address with TestUSD
cast rpc tempo_fundAddress 0x820525B56cF3cD426e58507F65DCaCB90BceEFb8 \
  --rpc-url https://${RPC_USERNAME}:${RPC_PASSWORD}@rpc-adagio.tempoxyz.dev

# Check your TestUSD balance
cast balance --erc20 0x20c0000000000000000000000000000000000000 \
  0x820525B56cF3cD426e58507F65DCaCB90BceEFb8 \
  --rpc-url https://${RPC_USERNAME}:${RPC_PASSWORD}@rpc-adagio.tempoxyz.dev
```
