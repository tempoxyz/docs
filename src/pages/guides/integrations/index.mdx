# Tempo Partner Integrations

Build on Tempo with our ecosystem of infrastructure partners.

## RPC & Infrastructure

Connect to Tempo with enterprise-grade RPC providers:

- [**Tempo RPC**](/guides/integrations/tempo-rpc) — Public endpoints and configuration
- [**QuickNode**](/guides/integrations/quicknode) — Enterprise RPC with advanced features
- [**Tenderly**](/guides/integrations/tenderly) — Debugging, simulation, and monitoring

## Wallets

Add wallet support to your Tempo application:

- [**MetaMask**](/guides/integrations/metamask) — Add Tempo network to MetaMask
- [**Privy**](/guides/integrations/privy) — Email, social login, and embedded wallets
- [**thirdweb**](/guides/integrations/thirdweb) — All-in-one SDK with wallet UI
- [**Turnkey**](/guides/integrations/turnkey) — Secure key infrastructure
- [**ZeroDev**](/guides/integrations/zerodev) — Smart accounts and gas sponsorship

## Smart Contracts

Deploy and interact with contracts:

- [**Foundry**](/guides/integrations/foundry) — Forge, Cast, and Anvil for Tempo
- [**Hardhat**](/guides/integrations/hardhat) — Deploy with Hardhat
- [**thirdweb**](/guides/integrations/thirdweb) — Pre-built contracts and deployment

## Interoperability

Bridge assets to and from Tempo:

- [**Bridge**](/guides/integrations/bridge) — Stablecoin orchestration
- [**Bridging Guide**](/guides/integrations/bridge) — Cross-chain transfers
